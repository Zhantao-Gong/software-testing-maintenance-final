# TrinityRCL Reproduction

This directory stores the TrinityRCL reproduction assets used in this course project.

Source:
- https://github.com/FireEverfly/Microservice-RCA-Experiment/tree/main/TrinityRCL

Included files:
- `collector.py`: Prometheus metric collection script
- `rca_v1.py` and `rca_v2.py`: root-cause analysis scripts
- `dependency.csv`, `fault_log.csv`, `metrics.csv`: example input data
- `rca_ranking.png`: example ranking output
