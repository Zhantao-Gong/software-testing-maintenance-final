# FluxEV Reproduction

This directory stores the FluxEV anomaly detection reproduction assets used in this course project.

Source:
- https://github.com/liaojunfan/fluxev-anomaly-detection-lab

Included contents:
- `src/`: FluxEV preprocessing and experiment scripts
- `fluxev_data/`: dataset, metadata, processed series, QA artifacts, and configuration files

Excluded from import:
- project PDF reports
- presentation files
- the original paper PDF bundled in the source repository
